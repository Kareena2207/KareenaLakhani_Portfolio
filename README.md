# Kareena Lakhani Portfolio

Static portfolio website for GitHub Pages.

## Files

- `index.html` - portfolio content and structure
- `styles.css` - responsive visual design
- `passport_photo_2x2.jpg` - portrait image

## Add Resume

Place the resume PDF in this folder and name it:

```text
Kareena_Lakhani_Resume.pdf
```

The Resume button in the website already points to that file.

## Deploy With GitHub Pages

1. Create a GitHub repository named `Kareena2207.github.io`.
2. Upload these project files to the repository root.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.

Your site will publish at:

```text
https://Kareena2207.github.io
```
